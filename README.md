# 射击靶子游戏

一个使用 Pygame 开发的射击靶子小游戏，考验你的反应速度和瞄准能力。

## 游戏特性

- 🎯 30秒限时挑战
- 🚀 靶子移动速度随时间递增
- 💥 击中靶子有粒子爆炸动画
- 🔊 开枪音效
- 🎨 美观的UI界面
- 🖱️ 准星光标

## 安装说明

### 环境要求
- Python 3.9 或更高版本
- pip 包管理器

### 安装依赖

```bash
pip install -r requirements.txt
```

或者手动安装 Pygame：

```bash
pip install pygame>=2.5.0
```

## 运行游戏

进入 `main` 目录并运行游戏：

```bash
cd main
python game.py
```

## 游戏说明

### 游戏流程

1. **开始界面**：显示游戏说明和"开始游戏"按钮
2. **游戏界面**：
   - 30秒倒计时
   - 屏幕上随机出现移动的靶子
   - 鼠标变为红色准星
   - 点击鼠标射击靶子
3. **结算界面**：显示击中靶子数量，可选择"再来一局"

### 游戏规则

- 游戏限时 **30秒**
- 同一时间只会出现 **一个靶子**
- 靶子会 **不规律地移动**
- 游戏越到后面，靶子移动 **速度越快**
- 击中靶子后会有碎开消失的过渡动画，随后在新位置出现新靶子
- 靶子未被击中时速度保持不变

### 操作说明

- 移动鼠标：控制准星瞄准
- 鼠标左键点击：开枪射击
- ESC键：退出游戏

## 模块说明

### ui.py
- `UI` 类：负责渲染开始界面、游戏界面、结算界面
- `Button` 类：按钮组件，支持悬停效果
- `Target` 类：靶子类，包含绘制和击中粒子动画

### control.py
- `GameControl` 类：处理鼠标事件，管理游戏操作
- `SoundManager` 类：生成并播放开枪音效

### move.py
- `Movement` 类：
  - 靶子随机位置生成
  - 不规律移动轨迹计算
  - 基于游戏时间的速度递增计算
  - 边界碰撞检测与反弹

## 技术特点

1. **多线程架构**：游戏逻辑与渲染分离，保证操作流畅性
2. **中文显示**：自动检测系统中文字体，避免乱码
3. **线程安全**：使用线程锁保护共享数据
4. **实时音效生成**：程序化生成开枪音效，无需外部音频文件
